# mj-divider

A horizontal rule styled like a CSS border.

## Key attributes

`border-color` (`#000000`), `border-style` (`solid`/`dashed`/`dotted`, default `solid`), `border-width` (`4px`), `width` (`100%`), `align` (`center`), `container-background-color`, `padding` (`10px 25px`).

```xml
<mj-divider border-width="1px" border-style="dashed" border-color="#e0e0e0" />
```
