# mj-spacer

Inserts vertical blank space between elements in a [column](/components/mj-column.md).

## Key attributes

`height` (`0px`), `container-background-color`, `padding`.

```xml
<mj-text>First line</mj-text>
<mj-spacer height="50px" />
<mj-text>Second line</mj-text>
```
