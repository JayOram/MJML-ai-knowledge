# Tooling

Authoring produces `.mjml`; these concepts cover turning it into HTML and catching mistakes.

- [CLI](/tooling/cli.md) — the `mjml` command-line compiler.
- [Node API](/tooling/node-api.md) — `mjml2html(input, options)`.
- [Validation](/tooling/validation.md) — `strict` / `soft` / `skip` levels.

Installation is via npm (`npm install mjml`). A hosted API (mjml.io/api) exists for environments that can't run Node.
