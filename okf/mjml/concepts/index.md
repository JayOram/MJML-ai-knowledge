# Concepts

The ideas here are prerequisites for using any [component](/components/index.md). Most MJML bugs are conceptual (a tag in the wrong slot), not syntactic.

- [Layout model](/concepts/layout-model.md) — the strict, shallow hierarchy and its five rules.
- [Ending tags](/concepts/ending-tags.md) — components that hold raw HTML but not nested MJML.
- [MJML 5 changes](/concepts/mjml-5-changes.md) — differences from v4 that cause regressions.
